Advanced SVG Path Designer

Path Data

The `d` attribute for your SVG path.

How to Use:

  • Add Point: Click on an empty area of the canvas.
  • Select Point: Click on an existing anchor point.
  • Move Point: Drag an anchor point or a control point handle.
  • Delete Point: Select a point and press 'Delete' or 'Backspace'.
  • Convert Point: Select a point (not the first one) to toggle it between a curve and a straight line corner.
  • Undo: Use the Undo button or press Ctrl+Z.
